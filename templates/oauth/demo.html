OAuth2.0 授权流程演示

1. 请求授权
→
2. 用户登录
→
3. 授权确认
→
4. 获取令牌
→
5. 获取用户信息

步骤1：请求授权

点击下面的按钮跳转到日志易平台进行授权：

前往日志易授权
{{ authorize_url }}

参数说明：

  • response_type=code：固定值，表示授权码模式
  • client_id：在日志易平台申请的客户端ID
  • redirect_url：必须与日志易平台配置一致
  • state：防CSRF攻击的随机字符串

步骤2-3：用户登录和授权

用户将在日志易平台完成以下操作：

  1. 输入用户名和密码登录日志易平台
  2. 查看应用请求的权限范围
  3. 点击"授权"按钮确认授权
注意：此步骤在日志易平台完成，当前演示应用无法直接控制。

步骤4：回调处理

授权成功后，日志易平台将重定向到：

{{ redirect_url }}?code=AUTHORIZATION_CODE&state=demo_state_123

当前演示应用将自动处理回调，交换访问令牌。

步骤5：获取用户信息

使用获取到的访问令牌调用用户信息接口：

GET {{ config.USERINFO_URL }}
Authorization: Bearer ACCESS_TOKEN
开始授权流程
或访问：返回首页
